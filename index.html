Usage

url encoding
request w/ json
A query for foil charging badger from born of the gods using json would look like this. json requests are made to /search 

{
    set: "Born of the Gods",
    name: "Charging badger",
    foil: true;
}
This api uses mtgjson to stay up to date. All set names are based off the name found there.

Limitations

The price api has several limitations:
Prices are refreshed hourly.
Paper prices only.
Sets' names my differ depending on the soure.
Some sets cannot be priced using this api.
TCG mid only
This api isn't finished yet.
Non-avaliable sets:
  • Guru
  • European Land Program
  • Asia Pacific Land Program
  • Multiverse Gift Box
  • Introductory Two-Player Set
  • Duels of the Planeswalkers
  • Any online only set
