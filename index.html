About

Up to date

This API uses webscrapping to obtain price data for MTG cards. The current scraping source is mtggoldfish, this means that this api is up to date as long as mtggoldfish is kept up to date.

Purpose

This price api was developed alongside an inventory system for MTG singles. While the inventoy system isn't a free service I've left this api open for anyone to use, as there is no easy to use and free price api. If usage becomes too high and hosting costs become to much the api will be closed.

Usage

url encoding
URL encoding is not supported yet.
request w/ json
A query for foil charging badger from born of the gods using json would look like this. json requests are made to /search 

{
    set: "Born of the Gods",
    name: "Charging badger",
    foil: true;
}
This api uses mtgjson's set names.

Limitations

The price api has several limitations:
  • Prices are refreshed hourly.
  • Paper prices only.
  • Sets' names my differ depending on the soure.
  • Some sets cannot be priced using this api.
  • TCG mid only
  • This api isn't finished yet.
Non-avaliable sets:
  • Guru
  • European Land Program
  • Asia Pacific Land Program
  • Multiverse Gift Box
  • Introductory Two-Player Set
  • Duels of the Planeswalkers
  • Any online only set
